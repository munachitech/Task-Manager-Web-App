Smart Task Manager Pro

A modern task management web application built with HTML, CSS, and JavaScript. The application helps users organize, prioritize, search, and track tasks through a clean and responsive interface.

Features

- Create, edit, and delete tasks
- Mark tasks as completed
- Priority levels (Low, Medium, High)
- Search functionality
- Filter tasks by All, Active, or Completed
- Dark mode support
- Data persistence using Local Storage
- Responsive user interface

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
